## 一、项目定位

### 1.1 一句话描述
一款基于飞虎队与第14航空队真实战役的俯视角纵向卷轴射击游戏，融合史实战役、机型演进与街机式清版射击体验。

### 1.2 核心体验目标
- **历史沉浸感**：每关对应真实战役/任务，地图、敌机、目标尽量贴合史实。
- **街机爽快感**：快节奏射击、多武器系统、BOSS战、炸弹清屏。
- **策略选择**：玩家可在多架战机间切换，每架战机有独特武器与炸弹。

### 1.3 参考游戏
- 《STRIKERS 1945》《STRIKERS 1945 II》（彩京）
- 《Raiden》（雷电）
- 《1942》《1943》（卡普空）

## 三、核心玩法循环

1. **选择战机**：每关开始前可选择已解锁战机。
2. **进入关卡**：地图自动卷轴，玩家控制飞机移动与射击。
3. **遭遇波次**：固定Y坐标触发敌机/地面目标刷出。
4. **摧毁目标**：击落敌机、炸毁地面目标获取分数与道具。
5. **BOSS战**：关卡末尾触发固定BOSS战。
6. **结算**：计算得分、击落数、无伤奖励，解锁下一关/隐藏关。

### 3.1 操作映射

| 操作 | 键位/手势 |
|------|----------|
| 移动 | WASD / 左摇杆 / 触屏拖拽 |
| 射击 | 自动连发，按住可蓄力 |
| 炸弹 | X / 手柄B键 / 触屏按钮 |
| 切换战机 | 关卡外机库界面 |

### 3.2 计分系统
- 击落敌机：100–5000分（按敌机类型）
- 摧毁地面目标：200–2000分
- 无伤通过波次：5000分
- 全关无伤：10000分
- 连续击落（Combo）：每10连击奖励500分

**历史背景**  
1941年12月20日，日军10架九七式重轰炸机空袭昆明。AVG P-40首次出战，击落9架日机，自己零损失。

- 中段：滇池上空
- 后段：昆明城区与滇缅公路
- 色调：云南高原、浅绿山丘、淡蓝湖水

**刷怪表**

**地面目标**

- 弱点：机腹炸弹舱

## 十三、AI开发工具提示

### 13.1 代码生成优先级

1. **核心框架**：Player、Enemy、Bullet、Bomb、GameManager 基类。
2. **关卡系统**：LevelManager，按Y坐标读取刷怪表。
3. **碰撞检测**：Area2D/CollisionShape2D 处理子弹、飞机、地面目标。
4. **地图卷轴**：Camera2D + 背景 Texture 垂直滚动。
5. **UI/HUD**：CanvasLayer 显示分数、生命、炸弹、BOSS血条。
6. **音效系统**：AudioStreamPlayer2D 管理射击、爆炸、BGM。
7. **存档系统**：通关状态、解锁战机、最高分。

### 13.2 数据驱动建议

- 每关配置使用独立 JSON/Resource 文件：
  - `levels/level_01.json`
  - `levels/level_02.json`
  - ...
- JSON包含：`background`、`scroll_speed`、`spawn_table`、`boss`、`weather`。
- 敌机参数使用独立 `enemy_data.json`。
- 玩家战机参数使用 `player_data.json`。

### 13.3 关键常量

```
SCREEN_WIDTH = 1080
SCREEN_HEIGHT = 1920
MAP_WIDTH = 800
MAP_HEIGHT = 2400
PLAYER_SPEED_BASE = 400
BULLET_SPEED_PLAYER = 1200
BULLET_SPEED_ENEMY = 300
SCROLL_SPEED_L01 = 16  # px/sec
```

## 十四、注意事项与限制

1. **历史准确性**：飞虎队实际未在中国战场与零式大规模交手，避免将零式作为常规敌机。
2. **敏感内容**：避免美化或重现盟军误炸平民事件（如六寨镇、汉口大轰炸平民伤亡）。
3. **广岛关处理**：以历史任务为背景，聚焦B-29技术与飞行挑战，避免 glorifying 核爆后果。
4. **文化尊重**：中日双方均作为战争参与者呈现，避免极端民族主义表达。
