# Flying Tigers 1945 — v1.5.0 升级设计文档

> **版本**: v1.5.0 Upgrade Plan v1.1
> **日期**: 2026-07-23
> **基准**: M3 完成状态（commit `aaa4e70`）
> **参考**: claude GDD v1.0（2026-07-22）+ 原始 design-spec + M3-F 军衔系统
> **核心目标**: 关卡设计贴合历史原型、保留舰艇 BOSS 作为不可击沉的中场 BOSS、引入陆上大型单位、丰富 BOSS 多样性

---

## 1. 升级哲学

v1.5.0 的核心原则是 **"历史沉浸优先，街机爽感不减"**：

1. **关卡重排**：按 1941~1945 时间线重排 10 个主线关，每关对应真实战役
2. **舰艇 BOSS 保留但改机制**：保留妙高/天龙/最上等真实舰艇作为"不可击沉中场 BOSS"，通过 assault_phase（限时摧毁炮塔）胜利
3. **陆上大型单位 BOSS**：增加筑垒要塞/弹药库/机场塔台/碉堡群/油库群等陆上大型目标作为 BOSS
4. **保留精华**：深渊模式、军衔系统、隐藏情报、对象池等 M3 成果全部保留
5. **玩法多样化**：引入炸桥/对舰/战略轰炸/基地防卫/护送等不同关卡类型
6. **多战机系统**：从单架 P-40 扩展为 7 架可选战机
7. **BOSS 无变身**：取消二阶段纹理切换（phase1→phase2），改为弱点暴露/召唤护航/多目标等更真实的 BOSS 机制
8. **震电改放入 H2 广岛**：作为 1945 日本本土防空战实际参战过的局地战斗机，J7W 震电在 H2 广岛作为终极拦截 BOSS 出现

## 3. BOSS 重设计方案（v1.1 修订版）

### 3.0 v1.1 重大变更

经 PM 与用户讨论后，对 v1.0 草案进行了三方面重大调整：

1. **舰艇 BOSS 全部保留**（不再"去舰艇化"），但改为**不可击沉的中场 BOSS**（assault_phase 机制）
2. **增加陆上大型单位 BOSS**（筑垒要塞/弹药库/机场塔台/碉堡群/油库群）
3. **震电改 J7W2 保留**，作为 H2 广岛的终极拦截 BOSS

### 3.1 核心机制变更

#### 3.1.1 取消 phase 变身

**旧机制**（v1.0~v1.4）：
- BOSS 有 2~3 个阶段，每个阶段切换纹理（phase1.png → phase2.png）
- 阶段切换触发变身动画 + 弹幕变化

**新机制**（v1.5.0）：
- BOSS **无纹理切换变身**，改为 **动态弱点系统 + 时间线性难度曲线**
- BOSS 全程保持同一外观，但 **弱点区域周期性暴露/切换**
- 弹幕随时间推移逐步增强（difficulty_curve 线性插值）
- 编队类 BOSS 可逐架击落（如 Ki-48 三机编队，击落一架后剩余两架弹幕加密）

#### 3.1.2 新增 assault_phase 机制（不可击沉 BOSS）

针对舰艇和大型陆上设施（弹药库、机场塔台）：

- BOSS 主体不可击沉/击毁，HP 设为 `int.MAX` 或 `indestructible: true` 标志
- BOSS 包含多个 **部件节点**（炮塔、防空炮、雷达、弹药堆），每个部件独立 HP
- **胜利条件**：限定时间（如 60~90s）内摧毁所有部件
- **失败条件**：玩家在 assault_phase 内死亡 → Game Over
- **退场动画**：所有部件摧毁后 BOSS 冒烟/起火减速离场，弹出 "AC-130 撤退" 提示

#### 3.1.3 BOSS 类型分类

| 类型 | 描述 | 击破条件 | 关卡 |
|------|------|---------|------|
| `formation` | 敌机编队（可击毁） | 击毁领头机/全部击毁 | L01, L09 |
| `ace` | 王牌单机（可击毁） | 直接击毁 | L04（机场段）, H1 |
| `naval_assault` | **不可击沉舰艇** | 限定时间内摧毁所有部件 | L02, L04（海峡段）, L08 |
| `ground_facility` | 地面大型设施 | 击毁主体 / 摧毁所有部件 | L03, L05, L06, L10 |
| `multi_target` | **多目标群**（编队/船队/碉堡群） | 依次击破 | L07, L10 |
| `mixed` | 空中+地面混合 | 分别击破 | L08, L10 |
| `final` | 终极 BOSS | 击毁主体 | H2 |
| `environmental` | 环境 BOSS | 击毁 + 环境配合 | H1 |

## 10. 风格决策：背景地图风格

新 GDD 提议复古航空地图风格（旧纸张底色+军用标注），与现有 orthophoto 写实风格冲突。

**PM 建议**：v1.5.0 **保留现有 orthophoto 写实风格**，理由：
1. M1~M3 已投入大量素材（95+ 张 PNG），全部替换成本过高
2. orthophoto 风格更贴合 iFighter 1945 参考风格
3. 新 GDD 的地图标注功能可通过 **HUD 叠加层** 实现（半透明路线/刷新点标注），无需改变背景本身

## 13. 总结

v1.5.0（v1.1 修订版）是一次 **历史还原 + BOSS 多元化 的版本升级**，核心改动量：

### Code 改动量
- 重构 `boss_base.gd`（去变身 + assault_phase + multi_target 三套机制）
- 重排 12→10 关卡配置
- 新增多战机系统/Combo/环境障碍/护送系统/assault_phase
- 2 个新 autoload（combo_manager.gd, escort_manager.gd）

### Design 改动量
- **14 个新 BOSS Sprite**（含 Ki-48 长机、天龙/妙高/最上舰艇、筑垒要塞、弹药库、塔台、震电改等）
- **3 套舰艇部件 Sprite**（炮塔/防空炮，约 30 个子部件图）
- **7 套陆上大型单位 Sprite**（要塞/弹药库/塔台/碉堡/油库/桥梁/烟囱）
- 7 个玩家战机 Sprite、3 个新关卡背景、3 个新敌机 Sprite、5 个地面目标 Sprite
- 6 张 BOSS 退场动画帧（舰艇冒烟/起火）

### 删除量
- 4 个旧 BOSS（akitsushima/yamato/yahata/shinden_final，shinden_final 重命名为震电改）
- ~30 张旧 BOSS Sprite（phase1/phase2/transform 全部删除）
- 2 种旧敌机（A6M/Ohka）
- 6 张旧背景

### 总计
- **新增素材约 75 张**
- **删除约 40 张**
- **代码改动涉及 boss_base.gd 核心重构 + 2 个新 autoload + 关卡配置全面更新**

**建议**: Code 和 Design 从 M4-A 开始并行推进。**Code 优先实现 assault_phase 机制**（B 部分 work item），**Design 优先绘制 3 套舰艇 Sprite**（天龙/妙高/最上），这两个是其他 BOSS 设计的基础。
