# Flying Tigers 1945 — v1.5.0 升级设计文档

> **版本**: v1.5.0 Upgrade Plan
> **日期**: 2026-07-23
> **基准**: M3 完成状态（commit `069dc8d`）
> **参考**: claude GDD v1.0（2026-07-22）+ 原始 design-spec + M3-F 军衔系统
> **核心目标**: 关卡设计贴合历史原型、BOSS 去舰艇化改为史实空战/地面目标、丰富关卡玩法多样性

---

## 1. 升级哲学

v1.5.0 的核心原则是 **"历史沉浸优先，街机爽感不减"**：

1. **关卡重排**：按 1941~1945 时间线重排 10 个主线关，每关对应真实战役
2. **BOSS 去舰艇化**：现有舰艇 BOSS（妙高/金刚/大和/翔鹤等）替换为史实敌机编队/地面设施
3. **保留精华**：深渊模式、军衔系统、隐藏情报、对象池等 M3 成果全部保留
4. **玩法多样化**：引入炸桥/对舰/战略轰炸/基地防卫/护送等不同关卡类型
5. **多战机系统**：从单架 P-40 扩展为 7 架可选战机
6. **BOSS 无变身**：取消二阶段纹理切换（phase1→phase2），改为弱点暴露/召唤护航/多目标等更真实的 BOSS 机制

## 3. BOSS 重设计方案

### 3.1 核心变更：取消变身机制

**旧机制**（v1.0~v1.4）：
- BOSS 有 2~3 个阶段，每个阶段切换纹理（phase1.png → phase2.png）
- 阶段切换触发变身动画 + 弹幕变化
- 需要 `phase_hps`、`phase_sprites`、`phase_bullets` 三组数据

**新机制**（v1.5.0）：
- BOSS **无纹理切换变身**，改为 **动态弱点系统**
- BOSS 全程保持同一外观，但 **弱点区域周期性暴露/切换**
- 弹幕随时间推移逐步增强（不用阶段切换，用时间线性插值）
- 编队类 BOSS 可逐架击落（如 Ki-48 三机编队，击落一架后剩余两架弹幕加密）

### 3.4 BOSS JSON 格式变更

旧格式（有 phase）：
```json
{
  "boss_id": "boss_bomber",
  "phase_hps": [150, 200],
  "phase_sprites": ["phase1.png", "phase2.png"],
  "phase_bullets": [["fan_shoot"], ["turret_fire", "fan_shoot"]]
}
```

新格式（v1.5.0，无 phase，用 difficulty_curve + weak_points）：
```json
{
  "boss_id": "boss_ki48_squadron",
  "boss_name": "Ki-48编队长机",
  "boss_type": "formation",  // formation / ace / ground / naval / mixed
  "max_hp": 3000,
  "sprite": "res://assets/sprites/boss/boss_ki48_squadron.png",
  "weak_points": [
    {"name": "bomb_bay", "region": {"x": -20, "y": 10, "w": 40, "h": 20}, "damage_mult": 2.0}
  ],
  "difficulty_curve": {
    "attack_interval_start": 1.5,
    "attack_interval_end": 0.6,
    "bullet_speed_mult_start": 1.0,
    "bullet_speed_mult_end": 1.8,
    "ramp_duration": 60.0
  },
  "bullet_patterns": ["fan_shoot", "aimed_shoot"],
  "summon": {"type": "ki27", "count": 2, "interval": 15.0},
  "contact_damage": 15,
  "drop_count": 3
}
```

## 6. 新增游戏系统

### 6.1 Combo 系统

- 连续击落计数器（0.5秒内连续击落累加）
- 每 10 连击奖励 500 分
- 被击中或超时清零
- HUD 中上位置显示 Combo 数

### 6.2 误击惩罚

- L07 湘江猎杀中的民船为中立目标
- 击中民船：扣 1000 分 + 取消 Combo + 屏幕闪红
- 实现方式：MapObject 增加 `faction` 属性（enemy/neutral/ally）

### 6.3 环境障碍系统

| 关卡 | 障碍 | 效果 |
|------|------|------|
| L03 怒江 | 峡谷山壁 | 撞壁即毁（StaticBody2D 碰撞） |
| L04 新竹 | 侧风 | 飞机受横向推力 |
| L05 衡阳 | 废墟 | 部分地面目标被毁后暴露隐藏高炮 |
| L06 宝庆 | 油库连锁 | 油库爆炸 AOE 连锁摧毁相邻目标 |
| L07 湘江 | 民船 | 中立目标，误击扣分 |
| L09 桂柳 | 喀斯特山峰 | 动态障碍，碰壁即毁 |
| H1 驼峰 | 强风+雷暴 | 横向推力 + 闪电伤害 |

## 10. 风格决策：背景地图风格

新 GDD 提议复古航空地图风格（旧纸张底色+军用标注），与现有 orthophoto 写实风格冲突。

**PM 建议**：v1.5.0 **保留现有 orthophoto 写实风格**，理由：
1. M1~M3 已投入大量素材（95+ 张 PNG），全部替换成本过高
2. orthophoto 风格更贴合 iFighter 1945 参考风格
3. 新 GDD 的地图标注功能可通过 **HUD 叠加层** 实现（半透明路线/刷新点标注），无需改变背景本身
